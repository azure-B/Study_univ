Jquery의 기본 이름표 달아주기

  • 1. HTML 요소에 id 혹은 class로 이름표를 달아준다
  • 2. Jquery로 이름표에 접근하여 제어한다
    • 제가 드린 이 코드의 내용을 본다면
    <img id="img">
    이런식으로 img 태그 안에 이름표로 id값이 img라고 들어있어요
    그 외 여러 div에 class가 선언되어 있을텐데 이도 마찬가지 입니다

    이렇게 HTML 요소 a나 div등 여러가지에 class 혹은 id값을 부여하는 것이 첫번째이며
    class와 id의 차이점은 id는 단 하나밖에 없어야 하며, class는 중복이 가능합니다.
